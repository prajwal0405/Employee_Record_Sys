# 👨‍💼 Employee Record System

A robust Java-based console application to manage employee records using Object-Oriented Programming (OOP), file handling, and database connectivity for efficient data management.

---

## 🚀 Overview

The Employee Record System is designed to perform CRUD operations on employee data through a clean, menu-driven interface. It demonstrates strong Java fundamentals including OOP principles, exception handling, database operations, and structured code design.

---

## ✨ Key Features

* ➕ Add new employee records
* 📋 View all employee details
* 🔍 Search employee by ID or name
* ✏️ Update employee information
* ❌ Delete employee records
* 💾 Database connectivity using JDBC
* ⚠️ Input validation and exception handling
* 📂 Organized code using multiple classes

---

## 🧠 Technical Highlights

* Clean separation of concerns using multiple classes
* OOP principles (Encapsulation, Classes, Objects)
* JDBC for database connectivity
* DAO pattern for handling database operations
* Exception handling for stable execution
* Scalable structure for future GUI or web integration

## ▶️ Getting Started

### Clone Repository

```bash
git clone https://github.com/prajwal0405/Employee_Record_Sys.git
```

### Run the Application

* Open the project in IntelliJ IDEA
* Configure MySQL database connection in `DBConnection.java`
* Run `Main.java`

## 📌 Sample Workflow

1. Add Employee → Enter employee details
2. View Employees → Display all records
3. Search Employee → Find by ID or Name
4. Update Employee → Modify existing data
5. Delete Employee → Remove employee record
